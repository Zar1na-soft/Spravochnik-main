Как заполнить прайс лист СК?

1. Прайс-листы

Функционал «Прайс-листы» предназначен для ввода в Систему электронных закупок поставщиками/производителями актуальных цен на поставляемую/производимую номенклатуру товаров, работ и/или услуг.

1.1. Добавление позиции ТРУ в прайс-лист в Системе путем ручного ввода данных

Шаг. 1.

Функционал добавления позиции ТРУ в прайс-листе доступен для пользователей с ролью системы «Участник закупок». Для добавления позиции в прайс-лист необходимо перейти в меню «Управление ценами» - «Прайс-лист»:

В случае наличия ранее добавленного прайс-листа, отобразится список позиций ТРУ в виде таблицы со столбцами

- Код ТРУ;

- Наименование ТРУ;

- Краткая характеристика ТРУ;

- Полная характеристика;

- Страна происхождения товара;

- Единица измерения;

- Текущая цена за единицу измерения (без НДС);

- Регион поставки;

- Статус поставщика;

- Возможный объем поставки;

- Производственная мощность в год;

- Дата опубликования;

- Статус;

- Действия.

Также есть возможность отфильтровать данный список с помощью кнопки

«Фильтры»:

Отобразится фильтрация (поиск) позиций ТРУ по следующим параметрам:

-Статус – выбор позиций ТРУ со статусом «Черновик», «Опубликовано», «На подписании» и «Архив»;

-Код ЕНС ТРУ – поиск кода ЕНС ТРУ по значению или поиск по дереву;

-Регион поставки – поиск региона поставки по значению или поиск по дереву;

-Единица измерения – выбор единицы измерения товара;

-Полная характеристика (согласно коду ЕНС ТРУ).

Кнопка «Найти» выведет список позиций ТРУ, согласно выбранным параметрам поиска, кнопка «Сбросить» очистит введенные данные.

Шаг 2.

Для того, чтобы создать новую позицию ТРУ необходимо нажать кнопку «Создать позицию ТРУ»:

После нажатия отобразится окно «Сведения о предмете прайс-листа», где требуется выбрать необходимый код ЕНС ТРУ:

Для удобства пользования, возможен поиск кода ЕНС ТРУ «по значению» и «по дереву»:


После выбора необходимого кода ЕНС ТРУ, отобразится окно с формой добавления позиции ТРУ в Прайс-лист.

В сведениях о предмете прайс-листа, системой автоматически указывается наименование товара/работы/услуги по ранее выбранному пользователем коду ЕНС ТРУ, его (её) краткая характеристика.


Имеется возможность поменять код ЕНС ТРУ.

Нажатие на пиктограмму , она осуществит возврат в окно выбора поиска кода ЕНС ТРУ

Дополнительная характеристика сведений о прайс-листе содержит обязательные к заполнению поля:

- «марка/модель/артикул товара»;

- «страна происхождения»;

- «завод/изготовитель товара».

Также можно добавить иную необходимую информацию, нажав кнопку «Добавить». Система отобразит дополнительные поля «Атрибут» (в этом поле можно отразить сведения, такие как: вес, размер, габариты, цвет, плотность, материал исполнения и прочее, не отраженные в краткой характеристике по данному коду ЕНС ТРУ), «Значение на русском», «Значение на казахском». Данные поля также являются обязательными для заполнения.

Для того, чтобы удалить добавленную строку необходимо нажать на пиктограмму X

В условиях по поставке товаров, работ и услуг указываются обязательные поля:

«Единица измерения», «Текущая цена за единицу измерения (без НДС)», «Валюта», «Регион поставки» и «Инкотермс».

Данные в поле «Валюта» заполняются пользователем в случае, если поставщик/производитель является нерезидентом. Имеется возможность выбора иностранных валют (доллар США, евро, российский рубль и др.). Система автоматически осуществит пересчет валюты в тенге по рыночному курсу Национального банка РК.

Есть возможность выбрать несколько регионов поставки, для этого необходимо нажать на пиктограмму "Добавить". Удаление осуществляется нажатием пиктограммы Х.

По мере необходимости заполняются поля «минимальный размер скидки в «%» - значение должно быть не меньше 0 и не должно превышать 100, и «Условия оплаты»(в этом поле можно отразить, к примеру, условияпо предоплате в «%-ном» выражении, по работе за наличный/безналичный расчет и т.п. ) - текст не более 500 символов.

В поле «Объемы поставляемых товаров, работ и услуг», необходимо заполнить обязательное поле «Статус поставщика» - выбрав из выпадающего списка «Производитель», «Поставщик», «Дилер/дистрибьютор».

Для статуса «Производитель» Система отобразит дополнительные обязательные для заполнения поля:

- «Производственная мощность в год».

- «Реестр» - на выбор из выпадающего списка «Отечественный товаропроизводитель» или «Товаропроизводитель иностранного государства».

- «Номер и дата выдачи сертификата» ( в случае, если пользователь – отечественный товаропроизводитель, система автоматически отобразит данные по номеру и дате выдачи сертификата из соответствующих информационных баз данных уполномоченного государственного органа, содержащих сведения об отечественных товаропроизводителях и производимых ими товарах).

- Также необязательное для заполнения поле «Примечание»:

Для статусов «Поставщик» или «Дилер/дистрибьютор», Система отобразит дополнительное обязательное для заполнения поле «Возможный объем поставки» и необязательное для заполнения поле «Примечание»:

Для сохранения введенных данных, необходимо нажать кнопку "Сохранить" . Для того, чтобы отменить создание позиции ТРУ, требуется нажать на кнопку "Отмена" .

Об успешном сохранении созданной позиции ТРУ сигнализирует сообщение в правом верхнем углу «Новый прайс успешно создан»:

1.2. Добавление позиции ТРУ в прайс-лист в Системе путем импорта прайс- листа

Шаг 1.

Необходимо нажать на кнопку «импорт»:

Система отобразит окно «Импорт прайс-листа».

Нажав на ссылку «XLS Шаблон импорта прайс-листа» можно загрузить шаблон прайс- листа.

Шаблон содержит поля для заполнения, аналогичные вводимым данным ручным способом в Систему.

Заполнив шаблон прайс-листа данными ТРУ пользователя, необходимо его импортировать в Систему, путем выбора «Загрузить шаблон».

Об успешной загрузке прайс-листа сигнализирует соответствующее сообщение Системы в правом верхнем углу.

В противном случае, система выдаст ошибки импорта шаблона, в виде таблицы с текстом ошибки (пример ошибок приведен ниже):

Для отмены импорта шаблона прайс-листа нажать на кнопку "Отмена"

1.3. Подписание и публикация прайс-листа

Загруженные в Систему позиции прайс-листа имеют статус «Черновик» и не доступны для просмотра другим пользователям.

Функционал подписания и публикации прайс-листа доступен пользователям с ролью в Системе «Утверждающий прайс-лист».

Для подписания и опубликования прайс-листа с помощью кнопки «Отправить на подписание» необходимо выбрать и отправить позиции ТРУ пользователю с ролью «Утверждающий прайс-лист».

Система уведомит сообщением в правом верхнем углу о том, что данные обновлены и статус позиций ТРУ прайс-листа изменится с «Черновик» на «На подписании»:

Далее, пользователю с ролью «Утверждающий прайс-лист» необходимо выбрать одну или несколько позиций ТРУ прайс-листа со статусом «На подписании» и нажать кнопку «Подписать и опубликовать». Система отобразит окно с выбором сертификата, пользователю потребуется подписать прайс-лист собственным ЭЦП.

Об успешном подписании и опубликовании позиций ТРУ прайс-листа, сигнализирует сообщение Системы в правом верхнем углу:

Статус автоматически сменится со статуса «На подписании» на «Опубликован»:

Также у пользователя с ролью «Утверждающего прайс-лист» есть возможность отправить одну или несколько позиций ТРУ со статусом «На подписании» на доработку, указав галочки по позициям и выбрав действие «Отправить на доработку»:


Система уведомит сообщением, что данные успешно обновлены:

1.4. Редактирование позиции ТРУ

Данный функционал доступен пользователям с ролью «Участник закупок» по позициям ТРУ со статусом «Черновик» и «Опубликован». Для этого требуется выбрать необходимую позицию ТРУ и действие «Редактировать»:

Система отобразит форму редактирования позиции ТРУ с ранее заполненными данными

После внесения изменений необходимо нажать на кнопку «Сохранить»,а для отмены выбрать кнопку «Отмена»:

1.5. Удаление позиции ТРУ

Данный функционал доступен пользователям с ролью «Участник закупок» «Утверждающий прайс-листы» в позициях ТРУ со статусом «Черновик». Для этого требуется выбрать необходимую позицию ТРУ и действие «Удалить»:

В подтверждение об удалении Система отобразит сообщение о подтверждении операции. При согласии требуется нажать кнопку «Да», а в противном случае на кнопку "Отмена"

Об успешном удалении будет сигнализировать сообщение Системы в правом верхнем углу:

1.6. Архивирование позиции ТРУ прайс-листа

Данный функционал доступен для пользователей с ролью «Участник закупки» или «Утверждающий прайс-лист» для позиций ТРУ со статусом «Опубликован».

Необходимо выбрать одну или несколько позиций ТРУ и выбрать действие «В архив»:

В подтверждение Система отобразит сообщение о подтверждении операции. При согласии, требуется нажать кнопку «Да», в противном случае, на кнопку Отмена

Об успешном удалении будет сигнализировать сообщение Системы в правом верхнем углу:

В случае добавления идентичной позиции в прайс-лист, Система автоматически отправит ранее добавленную позицию ТРУ со статусом «Опубликован» в архив.

1.7. Адресная рассылка

По опубликованным позициям прайс-листа, пользователю с ролью «Участник закупки» в личный кабинет системы поступит уведомление об объявленных организациями АО «Самрук-Қазына» закупках ТРУ и о публикации предварительного обсуждения проекта тендерной документации.

1.8. Уведомление о превышении/значительного занижения цены по прайс- листам.

При создании и редактировании прайс-листа, система осуществляет сравнение по данному коду ЕНС ТРУ с учетом единицы измерения и условий поставок ИНКОТЕРМС, цены по прайс-листу со средней ценой планов закупок организаций АО «Самрук-Казына». В случае превышения или значительного занижения, отобразится соответствующее предупреждение.

Данное сообщение не ограничивает сохранение и утверждение прайс-листа. Дополнительно, система отправит уведомление о превышении либо занижении.

Все позиции прайс-листа с превышениями/занижениями можно просмотреть в «Отчете о превышении/значительном занижении».
