Что такое KZTIN. Как получить


Чтобы продать товар через госзакупки поставщики размещают информацию о товаре в Электронном каталог товаров (ЭКТ). Регистрация товаров в нем бесплатная.

Чтобы продать товар через ЭКТ, посмотрите перечень товаров закупаемых через ЭКТ. Если ваш товар есть в этом перечне, то зарегистрируйтесь на сайте ЭКТ . Подайте заявку для размещения товара в каталоге.

Каждому зарегистрированному товару каталог присвоит уникальный код – KZTIN. При заполнении заявки на ЭКТ укажите GTIN - номер штрих кода на коробке товара, тогда каталог автоматически присвоит KZTIN. Загруженный в ЭКТ товар будет участвовать в процессе госзакупок. Веб-порталом госзакупок сформирует техспецификацию на товар автоматически из ЭКТ . После размещения товара в ЭКС, обязательно перепроверьте его регистрацию в каталоге.

Таким образом, процедура формирования и ведения ЭКТ осуществляют в подсистеме веб-портала государственных закупок по адресу https://e-catalog.gov.kz. В каталог размещают свои товары поставщики для продажи через госзакупки. Перечень товаров для ЭКТ утвержден МФ РК. При участии в госзакупках поставщики выбирают товары из ЭКТ. Техническую спецификацию по ним веб-портал формирует автоматически. Если товар входит в перечень товаров закупаемых их электронного каталога (ЭКТ), то заказчик будет выбирать товар, загруженный в ЭКТ (Статья 12 Закона РК О ГЗ).

Если товар входит в перечень товаров закупаемых их электронного каталога (ЭКТ), то заказчик будет выбирать товар, загруженный в ЭКТ (Статья 12 Закона РК О ГЗ). Пока эта функция не доступна заказчику.

Техническую спецификацию на товар веб-портал формирует из ЭКТ при подаче заявки для поставщика по госзакупкам конкурсом, аукционом, запросом ценовых предложений, из электронного магазина и из одного источника путем прямого заключения договора (п.72 Правил О ГЗ).

В техспецификации на товар с его описанием будет указан бренд, модель, страна происхождения товара. Заказчик будет выбирать товар по техспецификации, который ему подходит.

В перечне закупаемых из электронного каталога товаров:

- канцелярские товары,

- хозяйственные товары,

- лекарственные средства,

- изделия медицинского назначения,

- электротовары.

ЭКТ работает с 1 июля 2022 года, но пока только для поставщика.
